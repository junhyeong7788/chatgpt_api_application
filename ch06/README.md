# Bard API

바드API 깃허브 주소 - github.dsdanielpark/Bard-API

```
- pip install bardapi : 바드 설치
- __Secure-1PSID : 바드 api 키
```

크롬 바드 실행 -> 질문하나 함(api키 생성을 위해) -> F12 , 어플리케이션, 쿠키 -> 바드 -> Secure-1PSID 값을 복사
